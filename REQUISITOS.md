# Rquisitos - IRL Quest

## Requisitos Funcionales:
1. El sistema debe permitir al usuario crear un personaje con nombre, edad, peso y estatura.
2. El sistema debe crear las estadisticas por defecto del usuario en 0 (fuerza, velocidad, defensa, apariencia,        estamina, agilidad, carisma, inteligencia, flexibilidad, reflejos y determinación)
3. El sistema debe permitir al usuario crear misiones diarias pudiendo modificar lo siguiente: Que cosa hay que hacer para cumplir la misión, la dificultad de la misión, la experiencia y el oro que va a recibir dependiendo de la dificultad.
4. El sistema debe permitir al usuario crear misiones principales pudiendo modificar lo siguiente: Que cosa hay que hacer para cumplir la misión, la dificultad de la misión, la experiencia y el oro que va a recibir dependiendo de la dificultad y la fecha limite para cumnplir dicha misión.
5. El sistema debe usar la experiencia ganada en misiones para subir de nivel las estadisticas del usuario.
6. El sistema debe quitarle vida al personaje del usuario si no cumple con todas sus misiones diarias y/o no cumple con la misión principal en la fecha prevista.
7. El sistema debe permiter acumular el oro ganado por el usuario en la tienda del sistema.
8. El sistema debe permitirle al usuario poder comprar pociones de para restaurar la vida de su personaje.
9. El sistema debe "matar" al personaje del usuario cuandoo su vida llegue a 0.
10. El sistema debe crear eventos aleatorios cuando el usuario este interactuando con la app, estos eventos deben tener tiempo limite para completarlos.
11. El sistema va a crear un menu principal con el que el usuario pueda interactuar, el munu debe contener lo siguiente: Personaje, misiones diarias, misiones principales, cacerías aleatorias, tienda y un boton de guardado.

## Requisitos No Funcionales:
1. El sistema debe ejecutarse desde la terminal de Linux.
2. El sistema no debe de durar más de un minuto en abrirse.
3. El sistema debe ser escalable para en un futuro crear un aplicación movil.
4. El sistema debe aumentar el oro y la experiencia ganada por el usuario de manera inmediata.
5. El usuario debe de poder crear a su personaje con facilidad.
